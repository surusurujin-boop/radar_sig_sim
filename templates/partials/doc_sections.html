1. 프로젝트 개요

연구 목표

레이더 수신 환경에서 여러 방사원(emitter)의 펄스가 시간순으로 섞여 들어올 때, 각 펄스가 어느 방사원에 속하는지 자동으로 묶는 펄스 클러스터링을 수행합니다.

핵심 아이디어

  • PDW, IQ, Spectrum 세 가지 정보를 함께 사용
  • Multimodal Transformer로 펄스별 임베딩 학습
  • HDBSCAN 밀도 기반 클러스터링 (군집 수 K 사전 불필요)
합성 레이더 펄스 데이터 → 멀티모달 Transformer 임베딩 학습 → HDBSCAN 클러스터링

2. 개발 과정

  1. 문제 정의·설계 — 멀티모달 입력, Transformer, Contrastive + HDBSCAN
  2. 합성 데이터 — PRI/변조/SNR 시뮬레이터, IQ 순시·STFT 특징
  3. 모델 구현 — 3-Branch Encoder, Fusion, TOA Embedding
  4. 학습·평가 — Contrastive, Aux Loss, Baseline 비교
  5. Ablation — PDW / PDW+IQ / Full 정량 비교
  6. Flask 웹 — 대시보드, 학습, Explorer, DB
  7. 데이터 탐색기 — PDW/IQ/스펙트럼 시각화

11. 발표 체크리스트

  • 연구 문제: 다중 방사원 펄스 클러스터링
  • 입력: PDW(5) + IQ(256) + Spectrum(64×64)
  • 학습 데이터: S0, 3 emitters, train 80 / test 30
  • 모델: 3-Branch + Transformer + HDBSCAN
  • 핵심 결과: Full ARI 0.506 > PDW only 0.148
  • 설계 철학: PRI/DTOA 비사용, 펄스 내부 IQ·Spectrum 중심

13. 설계 철학

PRI/DTOA 같은 시계열 정보는 보조 문맥으로만 사용하고, 단일 펄스 내부의 변조(IQ)·주파수(Spectrum) 특성으로 클러스터링합니다.

DTOA-PRI baseline과 달리 제안 모델은 DTOA/PRI feature를 쓰지 않습니다. PDW+IQ Ablation(0.15→0.42)이 이 설계를 뒷받침합니다.

핵심 (펄스 내부)

  • IQ 3-Branch CNN
  • Spectrum 2D-CNN
  • Modulation Aux Loss

보조 (시계열/운용)

  • PDW Encoder (CF,PW,PA,DOA,TOA)
  • TOA Temporal Embedding
  • Transformer self-attention

vs DTOA-PRI Baseline

방법특징
DTOA-PRIDTOA 통계를 주 feature로 HDBSCAN
